# 哆啦A梦新番 视频播放网站源码

## 使用步骤  
  1. 配置好服务器环境。  
    - PHP  
  2. 下载源码并解压到你的网站根目录。  
  3. 搭建完成  
  4. 向资源目录里递送资源，PHP会自动生成对应资源的按钮。  
    - dora\src_dora\哆啦A梦新番  
    - dora\src_dora\哆啦A梦剧场版  
    - dora\src_dora\其他电影
